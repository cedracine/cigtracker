## 🚀 Install

1. Open **[cedracine.github.io/cigtracker](https://cedracine.github.io/cigtracker)** in Chrome or Safari
2. **Android** — Menu ⋮ → *Add to Home Screen*
3. **iOS** — Share ⬆ → *Add to Home Screen*
4. **Desktop** — install icon in Chrome/Edge address bar

Or self-host:
```bash
git clone https://github.com/cedracine/cigtracker.git
cd cigtracker
npx serve .   # or any static server
```

## 🛠 Tech Stack

- **Vanilla JS / HTML5 / CSS3** — zero dependencies
- **Chart.js** (CDN) for graphs
- **PWA** — Web App Manifest via Blob URL · Service Worker auto-unregistered on load (clean cache updates)
- **localStorage** for 100% local persistence
- **Bebas Neue + Courier Prime** (Google Fonts)

> ⚠ 1週間の実際の使用後、8日間のデモデータを削除して個人データのみ残しましょう。

2. 設定 ⚙ → 「日付範囲を削除」
3. 8日間のデモ日を入力 → 🗑 削除

> ★ インストール日をメモしておくと削除範囲を簡単に見つけられます。
